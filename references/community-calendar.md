# Community Calendar — Event Templates

## Event Types

### Recurring Events

**Weekly Meetup:**
```
📅 Weekly Community Meetup
🕐 Every Wednesday, 4:00 PM UTC
📍 #voice-chat / Online
📝 Open agenda — bring your questions, demos, or just hang out
```

**Monthly Hackathon:**
```
📅 Monthly Hackathon
🕐 Last Saturday of each month, 10:00 AM - 6:00 PM UTC
📍 #hackathon
🎯 Theme announced 1 week before
```

**Office Hours:**
```
📅 Office Hours with {expert}
🕐 Every {day}, {time} UTC
📍 #office-hours
📝 Pre-submit questions by replying to the announcement
```

### One-Time Events

**Workshop:**
```
📅 Workshop: {topic}
🕐 {date}, {time} UTC
📍 #workshops
👤 Led by: @{instructor}
📋 Prerequisites: {list}
🔢 Max participants: {limit}
```

**AMA:**
```
📅 AMA with {guest}
🕐 {date}, {time} UTC
📍 #ama
💡 About: {guest_bio}
📝 Submit questions early for priority
```

## Reminder Schedule

| Interval Before Event | Channel | Format |
|----------------------|---------|--------|
| 1 week | #events + #general | Full announcement with RSVP |
| 1 day | #general | Short reminder |
| 1 hour | #general + event channel | "Starting soon!" alert |
| At start | Event channel | "We're live!" with links |

## RSVP Tracking

Reactions on announcement:
- ✅ = Attending
- ❓ = Maybe
- 👀 = Interested but can't attend

Track: RSVP count vs actual attendance for future planning.

## Post-Event

1. Generate recap within 1 hour of event ending
2. Include: attendance, highlights, key takeaways
3. Archive Q&A content to knowledge base
4. Ask for feedback: "Rate this event 1-5"
5. Add to throwback calendar for future "on this day" posts
